# Alcance y Roles

## Dentro del Alcance ✓

- Gestión de usuarios con autenticación por rol
- Creación, edición y asignación de rutinas semanales (ejercicios, series, reps, videos)
- Vista de calendario para el cliente (ejercicios diarios)
- Registro de ejecución del cliente (peso, repeticiones, completado)
- Dashboard del entrenador con progreso y gráficas por cliente
- Registro de peso corporal semanal y fotos de progreso
- Registro de sesiones presenciales con observaciones del entrenador
- Test de 1RM con cálculo automático de carga por porcentaje
- Templates de rutina reutilizables para el entrenador
- App nativa Android

---

## Roles de Usuario

| Atributo | Entrenador | Cliente |
|---|---|---|
| Descripción | Profesional que gestiona múltiples clientes | Persona que sigue un plan de entrenamiento |
| Acceso | Cuenta única con acceso a todos sus clientes | Cuenta individual vinculada a su entrenador |
| Dispositivo | Android (teléfono o tablet) | Android (teléfono) |
| Nivel técnico | Básico-intermedio | Básico |
| Frecuencia de uso | Diaria / varias veces por semana | Diaria durante días de entrenamiento |
