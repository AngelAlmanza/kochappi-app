# Flujos Principales

## Entrenador: Crear y Asignar Rutina

1. Selecciona un cliente desde su lista
2. Accede a Rutinas y crea una nueva rutina semanal
3. Por cada día, agrega ejercicios: nombre, series, reps, video (opcional), modo de carga
4. Guarda y activa la rutina → el cliente recibe una notificación de nueva rutina disponible

## Cliente: Registrar Entrenamiento del Día

1. Abre la app → el calendario muestra los ejercicios del día actual
2. Selecciona el primer ejercicio; puede ver el video de referencia si está disponible
3. Por cada serie: ingresa peso y repeticiones, luego marca como completada
4. Repite para cada ejercicio → al terminar el último, el día queda marcado como completado

## Entrenador: Revisar Progreso de un Cliente

1. Ingresa al perfil del cliente
2. Accede a Progreso; selecciona rango de fechas o semana específica
3. Visualiza gráficas por ejercicio, cumplimiento de rutina y evolución del peso corporal
4. Navega a las fotos de progreso para evaluación visual
