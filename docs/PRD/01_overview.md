# Overview — Resumen Ejecutivo

Kochappi centraliza la relación entrenador-cliente en una app Android nativa:

- Asignación y seguimiento de rutinas de entrenamiento
- Registro de sesiones presenciales y autónomas
- Seguimiento del progreso físico (peso corporal, fotos, 1RM)
- Comunicación del desempeño entre entrenador y cliente

El sistema opera con **dos roles diferenciados**:
- **Entrenador** — planifica, supervisa y evalúa
- **Cliente** — ejecuta y registra

---

## Especificaciones Técnicas

| Dimensión | Descripción |
|---|---|
| Plataforma | Android API 26+ (Android 8.0+) |
| Usuarios | Entrenadores personales y sus clientes |
| Idioma inicial | Español |
| Distribución | APK privada |
| Conectividad | Internet para sincronización; offline parcial para entrenamientos |

---

## Objetivos de Negocio

- Digitalizar la operación del entrenador, eliminando papel y hojas de cálculo
- Incrementar la retención de clientes mediante visibilidad de su progreso
- Reducir el tiempo administrativo del entrenador en reportes manuales

## Objetivos de Usuario

**Entrenador**
- Diseñar y asignar rutinas semanales personalizadas por cliente
- Monitorear cumplimiento y progreso con datos en tiempo real
- Registrar observaciones durante sesiones presenciales

**Cliente**
- Acceder a su rutina semanal desde el teléfono de forma clara
- Registrar peso y repeticiones de cada ejercicio
- Ver su evolución física a través de fotos y datos de peso corporal
